# DRUM KIT

- Download the **Drums-release.zip** file from **releases** section
- Extract it
- Run the suitable file according to your OS.
  - **Windows** - *Drums-win_x64.exe*
  - **MacOS** - *Drums-mac_x64* 
  - **Linux**(x64) - *Drums-linux_x64*

Try it out here - https://m00ke5h.github.io/drum-pvt/
